Linux创建hexo博客上传到GitHub或者Gitee

Xshell连接虚拟机或者云服务器

1.下载vim编辑器

yum -y install vim

2.安装node.js

新建文件夹/usr/local/node
解压网上下的linux安装包 tar xvf node-v14.17.6-linux-x64.tar.xz

3.配置NODE系统环境变量

编辑 ~/.bash_profile ⽂件，在⽂件末尾追加如下信息:
#Nodejs
export PATH=/usr/local/node/node-v12.16.3-linux-x64/bin:$PATH

4.刷新环境变量，使之⽣效即可：

source ~/.bash_profile
检查安装结果
node -v
npm version
npx -v

5.使用淘宝镜像源（不嫌慢的可以不安装，我是1m带宽，安装了也慢，索性不安装）

npm install -g cnpm –registry=https://registry.npm.taobao.org
npm install cnpm@7.1.0 -g –registry=https://registry.npm.taobao.org (高版本报出“Error：Cannot find module ‘fs/promises”错误 降版本)
cnpm -v查看安装结果

6.全局安装hexo

cnpm install -g hexo-cli 或者 npm install -g hexo-cli

7.初始化博客，自建文件夹/usr/local/kolento/blog

mkdir blog
cd blog/
hexo init
npm install
启动
hexo s

8.更新博客

/usr/local/bolg/source/_posts
目录下博客内容，更新后记得重新生成静态文件
每次更新博客
hexo c # clean 清除缓存文件
hexo g # generate 生成静态文件。
hexo d # deploy 文件生成后部署

9.上传到github或者gitee

linux安装git插件
npm install –save hexo-deployer-git 或者 cnpm install –save hexo-deployer-git
修改blog目录中_config.yml文件
最下面修改为：（注意yml格式 冒号后有空一格）
#Deployment
##Docs: https://hexo.io/docs/one-command-deployment
deploy:
type: git
repo: https://github.com/KolentoMa/KolentoMa.github.io.git
branch: master

hexo d 上传到github

输入用户名和密码
xxxxxx
xxxxxx
部署成功
